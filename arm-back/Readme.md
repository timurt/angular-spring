# Apartment Rentals Management

Backend part

## Technologies
- Spring Boot 2.0.4
- PostgreSQL
- Java 8
- Jacoco 0.7.9
- Checkstyle 8.11

## Installation

Generate jacoco test coverage report
> $ mvn clean test jacoco:report

Run checkstyle verification
> $ mvn checkstyle:check

Run application
> $ mvn spring-boot:run
